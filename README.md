## Installation 

- Clone repository 

    $ git clone https://github.com/thuytpham/MH_Restful_API.git

- Create database

    $ createdb dbprograms 

- Activate the Virtual Environment 

    $ source env/bin/activate

- Install the dependencies 

    $ pip install -r requirements.txt

- Run the app

    $ python3 server.py 

- Run test_API.py

    $ python3 test_API.py
